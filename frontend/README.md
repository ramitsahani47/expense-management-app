# Expense Tracker Mobile App

A modern Expense Tracker mobile application built with React Native, TypeScript, and Expo.

## Features

* User Authentication
* Add Expense
* Update Expense
* Delete Expense
* View Expense List
* Form Validation
* Clean UI
* REST API Integration

## Tech Stack

### Frontend

* React Native
* Expo
* TypeScript
* React Navigation
* Axios

### Backend

* Node.js
* Express.js
* PostgreSQL
* TypeScript

## Project Structure


## Installation

```bash
git clone <repository-url>

cd expense-tracker

npm install

npx expo start
```
